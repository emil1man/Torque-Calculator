# Torque Calculator — Sprocket

[English](README.md) | [Русский](README_RU.md)

Калькулятор рекомендуемых характеристик двигателя для игры **Sprocket**. Инструмент помогает подобрать оптимальный крутящий момент и рабочий объём двигателя на основе массы танка и выбранной эпохи.

---

## 📋 Возможности

- **Калькулятор двигателя** — расчёт рекомендуемого крутящего момента и литража на основе удельной мощности и массы танка.
- **Характеристики эпох** — справочник с весом, мощностью, броней и вооружением для каждого класса танка по эпохам.
- **Поддержка тёмной темы** — встроенное переключение между светлой и тёмной темой (выбор сохраняется в `localStorage`).
- **Готовность к локализации** — i18n-инфраструктура для поддержки русского и английского языков.
- **Адаптивный дизайн** — интерфейс оптимизирован для мобильных устройств, планшетов и ПК.
- **Material Design** — использование Material Icons и современной палитры цветов.

---

## 📖 Использование

### Калькулятор двигателя

1. Перейдите в раздел «Калькулятор двигателя».
2. Выберите эпоху из выпадающего списка.
3. Введите целевую удельную мощность (HP/t).
4. Введите текущую массу танка (в тоннах).
5. Получите рекомендованные значения крутящего момента и объёма.

### Справочник эпох

1. Перейдите в раздел «Характеристики эпох».
2. Выберите эпоху из выпадающего списка.
3. На каждой карточке отображаются: вес, мощность двигателя, броня (корпус/башня) и калибр орудия.

---

## 🎨 Тёмная тема

Нажмите кнопку с иконкой луны/солнца в навбаре на любой странице для переключения темы. Выбор автоматически сохраняется в `localStorage`.

---

## 🌍 Многоязычность

### Структура переводов

Все строки интерфейса хранятся в JSON-файлах:
- `assets/i18n/ru.json` — русский
- `assets/i18n/en.json` — английский

---

## 📊 Данные эпох

Данные для справочника хранятся в файле `/data/lookups/basedata.json`.

---

## 🐛 Известные ограничения

- Данные эпох жёстко закодированы в `basedata.json` и `dTorque.json` (может потребоваться обновление вручную при изменении баланса игры).
- Fallback-данные для калькулятора встроены прямо в `data/calculator/index.html` (для работы в офлайн-режиме).
