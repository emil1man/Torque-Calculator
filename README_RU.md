# Torque Calculator — Sprocket

[English](README.md) | [Русский](README_RU.md)

Инструментарий для игры **Sprocket**, который помогает проектировать двигатели танков.

---

## 📋 Возможности

- **Калькулятор двигателя** — подбирает рекомендуемое количество цилиндров (n) и объём (V) для компактного двигателя, способного обеспечить заданную удельную мощность для танка.
- **Характеристики эпох** — справочник с ориентировочным весом, мощностью двигателя, бронированием (корпус/башня) и вооружением для лёгких, средних и тяжёлых танков по каждой эпохе.
- **Тёмная тема и локализация** — светлая/тёмная тема и интерфейс на русском/английском, оба сохраняются между сессиями.
- **Адаптивный дизайн** — работает на мобильных, планшетах и ПК.

---

## 📖 Использование

### Калькулятор двигателя

1. Перейдите на страницу **Калькулятор двигателя**.
2. Выберите эпоху из выпадающего списка.
3. Введите целевую удельную мощность (HP/t).
4. Введите текущую массу танка (в тоннах).
5. Получите рекомендуемые количество цилиндров (n) и объём одного цилиндра (V).
6. При желании откройте [**SprocketTools Gear Calculator**](https://sprockettools.github.io/TopGearCalculator.html) с предзаполненными значениями.

### Характеристики эпох

1. Перейдите на страницу **Характеристики эпох**.
2. Выберите эпоху из выпадающего списка.
3. На каждой карточке отображаются ориентировочные вес, удельная мощность, броня корпуса и башни, данные орудия — для лёгких, средних и тяжёлых танков.

## 🌍 Добавление нового языка

1. Создать `assets/i18n/xx.json` (взять `en.json` как шаблон).
2. Перевести все значения.
3. Добавить `'xx'` в массив `supportedLanguages` в файле `assets/js/lang.js`.
